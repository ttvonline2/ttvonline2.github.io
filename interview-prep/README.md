# Interview Prep Structure

This folder splits the interview preparation by topic.

How to use it:
- Open one topic folder at a time.
- Read the top-level `TOPIC_NAME_QA.md` file for the full question pool.
- Use each `qXX_...` folder to store small demos, code snippets, diagrams, notes, STAR stories, or company-specific talking points.

Recommended practice flow:
1. Read the short answer.
2. Add one project example from your experience.
3. Add one impact statement.
4. Save any supporting material inside the matching question folder.
